Comprendre les messages d'erreur de l'API vous permet de diagnostiquer rapidement les problèmes d'intégration.

## Codes HTTP Communs

| Code | Erreur | Cause Possible | Solution |
| :--- | :--- | :--- | :--- |
| `400` | Bad Request | Paramètre manquant ou mal formaté. | Vérifiez votre JSON par rapport à la documentation. |
| `401` | Unauthorized | Token manquant ou expiré. | Renouvelez votre Bearer Token ou vérifiez vos identifiants OAuth. |
| `403` | Forbidden | Droits insuffisants. | Vérifiez les permissions de votre Application ou de l'agent. |
| `404` | Not Found | Template ou Ressource inexistante. | Vérifiez le `message_template_id` ou l'ID de l'objet. |
| `429` | Too Many Requests | Limite de débit atteinte. | Réduisez la fréquence d'envoi ou vérifiez votre quota. |
| `500` | Server Error | Problème interne système. | Contactez le support Digishare si l'erreur persiste. |
