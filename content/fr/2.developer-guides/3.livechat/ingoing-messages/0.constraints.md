| Type           | Composant          | Limite          |
| :------------- | :----------------- | :-------------- |
| **Texte**      | `body`             | 4096 Caractères |
| **Interactif** | `header`           | 60 Caractères   |
| **Interactif** | Titre `button`     | 20 Caractères   |
| **Interactif** | Titre ligne `list` | 24 Caractères   |
