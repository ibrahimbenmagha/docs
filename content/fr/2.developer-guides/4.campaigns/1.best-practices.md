## 1. Variables Dynamiques

Utilisez systématiquement le format `%recipient.nomVariable%` dans votre objet `data` global.

- **Cohérence** : Assurez-vous que chaque variable définie dans le template a une valeur correspondante dans la liste des `contacts`.
- **Valeurs par défaut** : Si une variable est facultative dans votre logique métier, envoyez une chaîne vide plutôt que de l'omettre pour éviter les erreurs de parsing Meta.

## 2. Limites de Débit (Rate Limits)

::warning
Respectez les limites d'envoi de la WhatsApp Business API.
::

- **Échelle** : Commencez avec des volumes faibles pour construire votre réputation d'envoi (Tiering Meta).
- **Parallélisation** : Digishare gère la mise en file d'attente, mais évitez d'envoyer des milliers de requêtes individuelles simultanément ; privilégiez l'envoi groupé via le tableau `contacts`.

## 3. Maintenance des Templates

- **Approbation** : Seuls les templates au statut `APPROVED` dans votre dashboard Meta peuvent être envoyés.
- **Modifications** : Toute modification d'un template existant sur Meta nécessite une nouvelle période d'approbation avant de pouvoir être utilisée via l'API.
