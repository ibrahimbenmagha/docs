Ce guide montre comment envoyer une campagne WhatsApp avec un **header vidéo** et des variables dans le body.

::note
**Prérequis** :

- **Jeton Bearer** valide
- **ID de Modèle de Message** approuvé (par ex., `VOTRE_ID_MODELE`)
- **ID d'Étiquette d'Expéditeur** (par ex., `VOTRE_ID_EXPEDITEUR`)
::

## 1. Cas d'utilisation

Tutoriels, publicités vidéo, démonstrations de produits, messages événementiels.

## 2. Limitations vidéo

::warning
- Format **MP4 uniquement**
- Taille maximale : **16 MB**
- Durée recommandée : **moins de 60 secondes**
::

## 3. Payload

Le champ `header` contient l'objet `link` pointant vers la vidéo MP4.

::api-playground{method="POST" url="https://api.digishare.ma/v1/campaigns" description="Envoyez une campagne WhatsApp avec un header vidéo." :variables='{"token": "VOTRE_TOKEN", "senderId": "VOTRE_ID_EXPEDITEUR", "templateId": "VOTRE_ID_MODELE"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"title": "Tutoriel Vidéo", "message_type_id": "abnd5ym5dgkr7x4g", "message_template_id": "{templateId}", "contacts_and_data": {"data": {"wa_id": "%recipient.wa_id%", "third_id": "%recipient.third_id%", "header": {"link": "https://api.digishare.ma/storage/companies/digishare/company/80049/tutorial.mp4"}, "body1": "%recipient.body1%"}, "contacts": [{"wa_id": "212600000000", "body1": "Découvrez notre nouvelle fonctionnalité"}], "notify_webhooks": false, "meta": {}}, "_": {"channel": "whatsapp", "sender_label_id": "{senderId}", "conversation_id": "message-out"}}'}
::

## 4. Variables

| Variable | Description |
| :--- | :--- |
| `header.link` | URL publique de la vidéo (MP4) |
| `body1` | Variable de texte dans le corps du message |
