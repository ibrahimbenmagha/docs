Ce guide montre comment envoyer une campagne WhatsApp avec un **bouton Flow** (formulaire interactif).

::note
**Prérequis** :

- **Jeton Bearer** valide
- **ID de Modèle de Message** approuvé (par ex., `VOTRE_ID_MODELE`)
- **ID d'Étiquette d'Expéditeur** (par ex., `VOTRE_ID_EXPEDITEUR`)
::

## 1. Cas d'utilisation

Formulaires interactifs, collecte de données, wizards multi-étapes directement dans WhatsApp.

## 2. WhatsApp Flows

::tip
Les Flows permettent de créer des **expériences interactives complexes** (formulaires, sélections, etc.) sans quitter WhatsApp. Le Flow doit être créé et publié dans le **Business Manager** avant utilisation.
::
