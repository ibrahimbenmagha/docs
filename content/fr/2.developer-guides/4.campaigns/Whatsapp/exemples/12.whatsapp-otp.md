Ce guide montre comment envoyer une campagne WhatsApp de type **Authentication OTP** (One-Time Password).

::note
**Prérequis** :

- **Jeton Bearer** valide
- **ID de Modèle de Message** de catégorie **Authentication** approuvé
- **ID d'Étiquette d'Expéditeur** (par ex., `VOTRE_ID_EXPEDITEUR`)
::

## 1. Cas d'utilisation

Vérification d'identité, codes de connexion, validation de transactions, authentification à deux facteurs (2FA).

## 2. Sécurité OTP

::warning
- Les codes OTP ont généralement une durée de validité limitée (**5-10 minutes**)
- Utilisez uniquement des templates de catégorie **Authentication** pour les OTP
- Le bouton **Copy Code** permet à l'utilisateur de copier le code facilement
::

## 3. Payload

La variable `body1` contient le code OTP à envoyer.

::api-playground{method="POST" url="https://api.digishare.ma/v1/campaigns" description="Envoyez un code OTP via WhatsApp." :variables='{"token": "VOTRE_TOKEN", "senderId": "VOTRE_ID_EXPEDITEUR", "templateId": "VOTRE_ID_MODELE_AUTH"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"title": "Code Verification", "message_type_id": "abnd5ym5dgkr7x4g", "message_template_id": "{templateId}", "contacts_and_data": {"data": {"wa_id": "%recipient.wa_id%", "third_id": "%recipient.third_id%", "body1": "%recipient.body1%"}, "contacts": [{"wa_id": "212600000000", "body1": "123456"}], "notify_webhooks": false, "meta": {}}, "_": {"channel": "whatsapp", "sender_label_id": "{senderId}", "conversation_id": "message-out"}}'}
::

## 4. Variables

| Variable | Description |
| :--- | :--- |
| `body1` | Le code OTP à envoyer (ex: `123456`) |

::tip
Pour une meilleure sécurité, générez des codes aléatoires de **6 chiffres** et définissez une expiration courte dans votre backend.
::
