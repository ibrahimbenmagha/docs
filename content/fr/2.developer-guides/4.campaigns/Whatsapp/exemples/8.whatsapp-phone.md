Ce guide montre comment envoyer une campagne WhatsApp avec un **bouton d'appel téléphonique**.

::note
**Prérequis** :

- **Jeton Bearer** valide
- **ID de Modèle de Message** approuvé (par ex., `VOTRE_ID_MODELE`)
- **ID d'Étiquette d'Expéditeur** (par ex., `VOTRE_ID_EXPEDITEUR`)
::

## 1. Cas d'utilisation

Service client, prise de rendez-vous, support technique avec appel direct.

## 2. Configuration

::tip
Le numéro de téléphone est généralement configuré de manière **statique** dans le template sur le dashboard Meta. Il n'est pas passé dynamiquement via l'API.
::

## 3. Payload
