Ce guide montre comment envoyer une campagne WhatsApp avec un **header image** et des variables dans le body.

::note
**Prérequis** :

- **Jeton Bearer** valide
- **ID de Modèle de Message** approuvé (par ex., `VOTRE_ID_MODELE`)
- **ID d'Étiquette d'Expéditeur** (par ex., `VOTRE_ID_EXPEDITEUR`)
::

## 1. Cas d'utilisation

Promotions visuelles, annonces de produits, bannières marketing avec texte personnalisé.

## 2. Formats supportés

::tip
Images **JPG** et **PNG**. Taille recommandée : **800×418 pixels** (ratio 1.91:1) pour un affichage optimal.
::

## 3. Payload

Le champ `header` contient l'objet `link` pointant vers l'image. L'URL doit être **publiquement accessible**.

::api-playground{method="POST" url="https://api.digishare.ma/v1/campaigns" description="Envoyez une campagne WhatsApp avec un header image." :variables='{"token": "VOTRE_TOKEN", "senderId": "VOTRE_ID_EXPEDITEUR", "templateId": "VOTRE_ID_MODELE"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"title": "Promo Image", "message_type_id": "abnd5ym5dgkr7x4g", "message_template_id": "{templateId}", "contacts_and_data": {"data": {"wa_id": "%recipient.wa_id%", "third_id": "%recipient.third_id%", "header": {"link": "https://api.digishare.ma/storage/companies/digishare/company/16/banner.jpg"}, "body1": "%recipient.body1%"}, "contacts": [{"wa_id": "212600000000", "body1": "Promo été 2026 - 50% de réduction"}], "notify_webhooks": false, "meta": {}}, "_": {"channel": "whatsapp", "sender_label_id": "{senderId}", "conversation_id": "message-out"}}'}
::

## 4. Variables

| Variable | Description |
| :--- | :--- |
| `header.link` | URL publique de l'image (JPG/PNG) |
| `body1` | Variable de texte dans le corps du message |

::warning
Assurez-vous que l'URL de l'image est **publiquement accessible**. Testez-la dans un navigateur en mode incognito avant l'envoi.
::
