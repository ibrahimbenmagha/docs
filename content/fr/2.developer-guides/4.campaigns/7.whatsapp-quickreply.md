Ce guide montre comment envoyer une campagne WhatsApp avec des **boutons Quick Reply**.

::note
**Prérequis** :

- **Jeton Bearer** valide
- **ID de Modèle de Message** approuvé (par ex., `VOTRE_ID_MODELE`)
- **ID d'Étiquette d'Expéditeur** (par ex., `VOTRE_ID_EXPEDITEUR`)
::

## 1. Cas d'utilisation

Sondages, confirmations (Oui/Non), choix d'options, feedback client.

## 2. Actions automatiques

::tip
Vous pouvez configurer des actions pour chaque réponse : envoyer un message, créer un ticket, ajouter un tag, déplacer la conversation, etc.
::

## 3. Payload

Les boutons Quick Reply sont définis dans les champs `button0`, `button1`, etc. Maximum **3 boutons** (jusqu'à 10 avec les règles avancées).

## 4. Variables

| Variable | Description |
| :--- | :--- |
| `button0` | Texte du premier bouton Quick Reply |
| `button1` | Texte du deuxième bouton Quick Reply |
| `button2` | Texte du troisième bouton (optionnel) |

::tip
Pour plus de détails sur les combinaisons de boutons autorisées, consultez les [Règles des Boutons](/fr/developer-guides/button-rules).
::
