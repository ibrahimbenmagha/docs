Ce guide montre comment envoyer une campagne WhatsApp avec un **carrousel** (multi-cartes défilables).

::note
**Prérequis** :

- **Jeton Bearer** valide
- **ID de Modèle de Message** approuvé (par ex., `VOTRE_ID_MODELE`)
- **ID d'Étiquette d'Expéditeur** (par ex., `VOTRE_ID_EXPEDITEUR`)
::

## 1. Cas d'utilisation

Envoyez un message avec **plusieurs cartes défilables** — parfait pour présenter des produits, des services ou des options que les utilisateurs peuvent parcourir horizontalement.

- **Catalogues et vitrines de produits** — Présentez plusieurs articles en un seul message
- **Forfaits et options de services** — Comparez différentes offres côte à côte
- **Galeries ou menus d'événements** — Affichez des images avec descriptions
- **Promotions multiples** — Deals, nouveautés dans un seul message

## 2. Règles importantes

::warning
- **Minimum 2 cartes, maximum 10**
- Chaque carte nécessite une **image ou une vidéo** (header média obligatoire)
- Toutes les cartes doivent avoir la **même structure** (même nombre de boutons, même type)
- Maximum **2 boutons par carte**
::

## 3. Structure des variables

| Variable | Description | Exemple |
| :--- | :--- | :--- |
| `card{N}_header` | Image/Vidéo de la carte N (commence à 0) | `card0_header`, `card1_header` |
| `card{N}_body{M}` | Variable M du body de la carte N | `card0_body1`, `card1_body1` |
| `card{N}_button{M}` | Bouton M de la carte N | `card0_button0`, `card1_button0` |

::tip
Utilisez des images de **haute qualité** (ratio 1:1 recommandé). Gardez le texte des boutons court et actionnable.
::
