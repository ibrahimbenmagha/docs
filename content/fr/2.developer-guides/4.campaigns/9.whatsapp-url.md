Ce guide montre comment envoyer une campagne WhatsApp avec un **bouton URL**.

::note
**Prérequis** :

- **Jeton Bearer** valide
- **ID de Modèle de Message** approuvé (par ex., `VOTRE_ID_MODELE`)
- **ID d'Étiquette d'Expéditeur** (par ex., `VOTRE_ID_EXPEDITEUR`)
::

## 1. Cas d'utilisation

Redirection vers un site web, page de paiement, formulaire, suivi de commande.

## 2. URL dynamique

::tip
Vous pouvez utiliser une URL de base statique + une variable. Exemple : `www.monsite.com/{{1}}` où `{{1}}` sera remplacé par une valeur dynamique (ex: un ID de commande).
::
