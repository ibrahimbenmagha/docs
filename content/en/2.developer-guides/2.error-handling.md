Understanding API error messages allows you to quickly diagnose integration issues.

## Common HTTP Codes

| Code | Error | Possible Cause | Solution |
| :--- | :--- | :--- | :--- |
| `400` | Bad Request | Missing or malformed parameter. | Check your JSON against the documentation. |
| `401` | Unauthorized | Missing or expired token. | Renew your Bearer Token or check your OAuth credentials. |
| `403` | Forbidden | Insufficient rights. | Check your Application or agent permissions. |
| `404` | Not Found | Template or Resource does not exist. | Check the `message_template_id` or object ID. |
| `429` | Too Many Requests | Rate limit reached. | Reduce sending frequency or check your quota. |
| `500` | Server Error | Internal system issue. | Contact Digishare support if the error persists. |
