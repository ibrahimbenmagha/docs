Message templates are pre-approved by Meta and classified into three distinct categories based on their purpose. Each category has its own usage rules and pricing.

## Template Categories

### 📣 Marketing
Marketing templates are used to promote products or services, send promotional offers, or re-engage with customers.
- **Function**: Sales, follow-ups, promotions, invitations.
- **Indicator**: Any message containing promotional content or that does not fit into the Utility or Authentication categories.

### 🛠️ Utility
These messages facilitate a specific transaction or request that is already in progress.
- **Function**: Order confirmations, delivery notifications, invoices, account statements.
- **Indicator**: Content related to a user action or an update to an existing transaction.

### 🔐 Authentication
Used to secure user account access.
- **Function**: Sending verification codes (OTP), account recovery.
- **Indicator**: Message containing only a security code and minimal instructions.
