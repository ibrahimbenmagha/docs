This guide shows how to send a WhatsApp campaign with a **phone call button**.

::note
**Prerequisites**:

- Valid **Bearer Token**
- Approved **Message Template ID** (e.g., `YOUR_TEMPLATE_ID`)
- **Sender Label ID** (e.g., `YOUR_SENDER_ID`)
::

## 1. Use Cases

Customer service, appointment booking, technical support with direct calling.

## 2. Configuration

::tip
The phone number is typically configured **statically** in the template on the Meta dashboard. It is not passed dynamically via the API.
::

## 3. Payload

::api-playground{method="POST" url="https://api.digishare.ma/v1/campaigns" description="Send a WhatsApp campaign with a phone call button." :variables='{"token": "YOUR_TOKEN", "senderId": "YOUR_SENDER_ID", "templateId": "YOUR_TEMPLATE_ID"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"title": "Contact Support", "message_type_id": "abnd5ym5dgkr7x4g", "message_template_id": "{templateId}", "contacts_and_data": {"data": {"wa_id": "%recipient.wa_id%", "third_id": "%recipient.third_id%"}, "contacts": [{"wa_id": "212600000000"}], "notify_webhooks": false, "meta": {}}, "_": {"channel": "whatsapp", "sender_label_id": "{senderId}", "conversation_id": "message-out"}}'}
::

::warning
Maximum **1 PHONE_NUMBER button** per template. See [Button Rules](/en/developer-guides/button-rules) for allowed combinations.
::
