This guide shows how to send a WhatsApp campaign with a **carousel** (scrollable multi-cards).

::note
**Prerequisites**:

- Valid **Bearer Token**
- Approved **Message Template ID** (e.g., `YOUR_TEMPLATE_ID`)
- **Sender Label ID** (e.g., `YOUR_SENDER_ID`)
::

## 1. Use Cases

Send a message with **multiple scrollable cards** — perfect for showcasing products, services, or options users can browse horizontally.

- **Product catalogs** — Present multiple items in a single message
- **Service plans** — Compare different offers side by side
- **Event galleries** — Display images with descriptions
- **Multiple promotions** — Deals, new arrivals in one message

## 2. Important Rules

::warning
- **Minimum 2 cards, maximum 10**
- Each card requires an **image or video** (media header mandatory)
- All cards must have the **same structure** (same number of buttons, same type)
- Maximum **2 buttons per card**
::

## 3. Variable Structure

| Variable | Description | Example |
| :--- | :--- | :--- |
| `card{N}_header` | Image/Video of card N (starts at 0) | `card0_header`, `card1_header` |
| `card{N}_body{M}` | Variable M of card N body | `card0_body1`, `card1_body1` |
| `card{N}_button{M}` | Button M of card N | `card0_button0`, `card1_button0` |

::tip
Use **high quality** images (1:1 ratio recommended). Keep button text short and actionable.
::
