This guide shows how to send a WhatsApp campaign with an **image header** and body variables.

::note
**Prerequisites**:

- Valid **Bearer Token**
- Approved **Message Template ID** (e.g., `YOUR_TEMPLATE_ID`)
- **Sender Label ID** (e.g., `YOUR_SENDER_ID`)
::

## 1. Use Cases

Visual promotions, product announcements, marketing banners with personalized text.

## 2. Supported Formats

::tip
**JPG** and **PNG** images. Recommended size: **800×418 pixels** (1.91:1 ratio) for optimal display.
::

## 3. Payload

The `header` field contains the `link` object pointing to the image. The URL must be **publicly accessible**.

::api-playground{method="POST" url="https://api.digishare.ma/v1/campaigns" description="Send a WhatsApp campaign with an image header." :variables='{"token": "YOUR_TOKEN", "senderId": "YOUR_SENDER_ID", "templateId": "YOUR_TEMPLATE_ID"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"title": "Image Promo", "message_type_id": "abnd5ym5dgkr7x4g", "message_template_id": "{templateId}", "contacts_and_data": {"data": {"wa_id": "%recipient.wa_id%", "third_id": "%recipient.third_id%", "header": {"link": "https://api.digishare.ma/storage/companies/digishare/company/16/banner.jpg"}, "body1": "%recipient.body1%"}, "contacts": [{"wa_id": "212600000000", "body1": "Summer Promo 2026 - 50% off"}], "notify_webhooks": false, "meta": {}}, "_": {"channel": "whatsapp", "sender_label_id": "{senderId}", "conversation_id": "message-out"}}'}
::

## 4. Variables

| Variable | Description |
| :--- | :--- |
| `header.link` | Public image URL (JPG/PNG) |
| `body1` | Text variable in the message body |

::warning
Make sure the image URL is **publicly accessible**. Test it in an incognito browser before sending.
::
