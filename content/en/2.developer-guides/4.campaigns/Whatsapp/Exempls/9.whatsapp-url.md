This guide shows how to send a WhatsApp campaign with a **URL button**.

::note
**Prerequisites**:

- Valid **Bearer Token**
- Approved **Message Template ID** (e.g., `YOUR_TEMPLATE_ID`)
- **Sender Label ID** (e.g., `YOUR_SENDER_ID`)
::

## 1. Use Cases

Website redirect, payment page, forms, order tracking.

## 2. Dynamic URL

::tip
You can use a static base URL + a variable. Example: `www.mysite.com/{{1}}` where `{{1}}` will be replaced by a dynamic value (e.g., an order ID).
::
