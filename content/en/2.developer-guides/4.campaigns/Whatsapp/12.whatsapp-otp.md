This guide shows how to send a WhatsApp **Authentication OTP** (One-Time Password) campaign.

::note
**Prerequisites**:

- Valid **Bearer Token**
- Approved **Authentication** category **Message Template ID**
- **Sender Label ID** (e.g., `YOUR_SENDER_ID`)
::

## 1. Use Cases

Identity verification, login codes, transaction validation, two-factor authentication (2FA).

## 2. OTP Security

::warning
- OTP codes typically have a limited validity period (**5-10 minutes**)
- Only use **Authentication** category templates for OTP
- The **Copy Code** button allows the user to easily copy the code
::

## 3. Payload

The `body1` variable contains the OTP code to send.

::api-playground{method="POST" url="https://api.digishare.ma/v1/campaigns" description="Send an OTP code via WhatsApp." :variables='{"token": "YOUR_TOKEN", "senderId": "YOUR_SENDER_ID", "templateId": "YOUR_AUTH_TEMPLATE_ID"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"title": "Verification Code", "message_type_id": "abnd5ym5dgkr7x4g", "message_template_id": "{templateId}", "contacts_and_data": {"data": {"wa_id": "%recipient.wa_id%", "third_id": "%recipient.third_id%", "body1": "%recipient.body1%"}, "contacts": [{"wa_id": "212600000000", "body1": "123456"}], "notify_webhooks": false, "meta": {}}, "_": {"channel": "whatsapp", "sender_label_id": "{senderId}", "conversation_id": "message-out"}}'}
::

## 4. Variables

| Variable | Description |
| :--- | :--- |
| `body1` | The OTP code to send (e.g., `123456`) |

::tip
For better security, generate random **6-digit codes** and set a short expiration in your backend.
::
