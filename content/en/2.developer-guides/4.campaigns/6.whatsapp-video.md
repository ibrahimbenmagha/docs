This guide shows how to send a WhatsApp campaign with a **video header** and body variables.

::note
**Prerequisites**:

- Valid **Bearer Token**
- Approved **Message Template ID** (e.g., `YOUR_TEMPLATE_ID`)
- **Sender Label ID** (e.g., `YOUR_SENDER_ID`)
::

## 1. Use Cases

Tutorials, video ads, product demos, event messages.

## 2. Video Limitations

::warning
- **MP4 format only**
- Maximum size: **16 MB**
- Recommended duration: **under 60 seconds**
::

## 3. Payload

The `header` field contains the `link` object pointing to the MP4 video.

::api-playground{method="POST" url="https://api.digishare.ma/v1/campaigns" description="Send a WhatsApp campaign with a video header." :variables='{"token": "YOUR_TOKEN", "senderId": "YOUR_SENDER_ID", "templateId": "YOUR_TEMPLATE_ID"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"title": "Video Tutorial", "message_type_id": "abnd5ym5dgkr7x4g", "message_template_id": "{templateId}", "contacts_and_data": {"data": {"wa_id": "%recipient.wa_id%", "third_id": "%recipient.third_id%", "header": {"link": "https://api.digishare.ma/storage/companies/digishare/company/80049/tutorial.mp4"}, "body1": "%recipient.body1%"}, "contacts": [{"wa_id": "212600000000", "body1": "Discover our new feature"}], "notify_webhooks": false, "meta": {}}, "_": {"channel": "whatsapp", "sender_label_id": "{senderId}", "conversation_id": "message-out"}}'}
::

## 4. Variables

| Variable | Description |
| :--- | :--- |
| `header.link` | Public video URL (MP4) |
| `body1` | Text variable in the message body |
