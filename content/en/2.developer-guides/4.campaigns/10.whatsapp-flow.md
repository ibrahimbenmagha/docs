This guide shows how to send a WhatsApp campaign with a **Flow button** (interactive form).

::note
**Prerequisites**:

- Valid **Bearer Token**
- Approved **Message Template ID** (e.g., `YOUR_TEMPLATE_ID`)
- **Sender Label ID** (e.g., `YOUR_SENDER_ID`)
::

## 1. Use Cases

Interactive forms, data collection, multi-step wizards directly in WhatsApp.

## 2. WhatsApp Flows

::tip
Flows allow you to create **complex interactive experiences** (forms, selections, etc.) without leaving WhatsApp. The Flow must be created and published in the **Business Manager** before use.
::

## 3. Payload

Maximum **1 FLOW button** per template. Flow buttons cannot be combined with Quick Reply buttons. See [Button Rules](/en/developer-guides/button-rules).
